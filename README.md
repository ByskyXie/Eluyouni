# Eluyouni（医路有你）

## 简介

第八届三创赛（创新、创意、创业）参赛作品，江西云惠医健康咨询有限公司官方APP。

“医路有我”APP是一款以知识分享、信息共享为核心的受控自媒体平台。其以中老年慢性病患者为初期目标用户群体，创新性采用竞争性分级免费策略，以激励用户通过“病毒式营销”方式推广APP平台，激励用户分享更有价值的治疗故事，激励医生不断提高治疗水平、分享高质量的医疗处方和治疗经历等医疗知识，从而将平台打造成具备医疗咨询、病友交流、医生交流、病例评析、医疗方案制定、寻医问药、会诊室等主要功能的互联网+APP医疗知识生态圈。

公司致力于构建以知识分享、信息共享的受控自媒体平台为核心，采用创新性运营模式，秉承“平等、学习、创新和尊重知识”的理念，努力促进医疗资源共享，创新医疗服务模式，改善用户健康，提升人类幸福。
